Options de compte

Dans la section Options du compte, vous pouvez administrer tous les aspects de votre expérience bancaire avec les banques QBCore, de commander une nouvelle carte de débit, de modifier votre code PIN et même de créer un compte d'épargne.

Gestion des cartes de débit

Une carte de débit peut être commandée pour vous permettre d'utiliser des distributeurs automatiques de billets et de payer les caractéristiques de la carte fournie gratuitement par QBCore Management.

Pour demander une nouvelle carte de débit, vous devez d'abord établir un numéro d'identification personnel (PIN) Cela vous permet d'utiliser votre carte chez les distributeurs automatiques de billets

Gestion des comptes d'épargne

Vous n'avez pas encore ouvert de compte d'épargne avec QBCore Banks, cliquez sur le bouton pour ouvrir un compte d'épargne personnel, avec un numéro de compte séparé et un code de classification.
